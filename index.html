My Work

Store Billing System

I developed a store billing system using Python and Tkinter for the user interface. The system enables efficient transaction management, inventory tracking, and invoice generation. Tkinter provides a clean and interactive interface, while Python handles backend logic, including real-time inventory updates. The system enhances the overall store operation by streamlining billing and inventory management.

Simple Weather App

I crafted a straightforward weather app using Java's Swing for the user interface and JDBC for database connectivity. The Swing framework facilitated the creation of a user-friendly interface with components like buttons and text fields. Users can input a location, and the app retrieves weather data from the connected database using JDBC.

Guess the Number

My inaugural project as a web developer was a "Guess My Number" app, marked by its simplicity and beginner-friendly syntax. Leveraging HTML, CSS, and JavaScript, I created an interactive web-based game. The HTML structure defined the layout, while CSS added a touch of styling to make the interface visually appealing. JavaScript played a pivotal role in implementing the game logic. Using basic syntax and control flow statements, I prompted users to input their guessed number and provided feedback on whether the guess was too high, too low, or correct. The app also incorporated a limited number of attempts, enhancing the challenge for players.
